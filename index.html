      Roadmap

      The first path should start with Shopify orientation, then move into local theme workflow.

      1. Dashboard Tour

      Learn where orders, products, customers, content, analytics, discounts, and settings live.

      2. Store Setup

      Create the account, confirm business settings, add staff access, and understand billing basics.

      3. Online Store

      Add pages, menus, products, collections, and customize the theme before launch QA.

      4. VS Code and CLI

      Install VS Code, Node, npm, Git, Shopify CLI, then preview and check theme work locally.

      5. GitHub Desktop

      Confirm supervisor access, clone the assigned repo, pull first, commit clearly, push, and ask for review.

      6. BEM and SCSS

      Learn naming conventions, SCSS nesting, ampersand modifiers, architecture, and CSS review standards.

      Glossary

      Terms beginners should know before changing the store.

      AdminThe private Shopify workspace for managing the store.
      StorefrontThe customer-facing website shoppers see.
      ThemeThe design system controlling storefront layout and style.
      SectionA configurable content area inside the theme editor.
      Shopify BlockA smaller editable item inside a section in the theme editor.
      ProductAn item for sale, including price, media, inventory, and status.
      CollectionA group of products used for browsing and merchandising.
      Sales channelA place where products are sold, like Online Store.
      VS CodeThe code editor used to open project folders and run terminal commands.
      npmThe package manager used to install project dependencies and Shopify CLI.
      Shopify CLIThe command-line tool used to preview, pull, check, and push Shopify themes.
      Theme devA Shopify CLI preview command for testing local theme changes.
      Gulp watchA project build command that watches source files and compiles assets when configured.
      RepositoryA GitHub project that stores code, history, branches, and collaboration work.
      CloneA local copy of a GitHub repository on the trainee computer.
      CommitA saved local checkpoint with a message explaining the code change.
      PushUploading local commits to GitHub for review or collaboration.
      BEMA CSS naming convention using block, element, and modifier names.
      BEM BlockA standalone UI component used in CSS naming, such as product-card or site-header.
      ElementA dependent part of a block, written like block__element.
      ModifierA variation of a block or element, written like block--modifier.
      SCSSSass syntax that looks like CSS and compiles into normal CSS.
      &The SCSS parent selector used for BEM patterns like &__title and &--active.
