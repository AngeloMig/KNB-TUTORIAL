Screenshot asset naming

Guided popup screenshots are loaded from:

assets/shopify/{module-key}-{step-name}.png

Examples:
- assets/shopify/dashboard-home.png
- assets/shopify/dashboard-orders.png
- assets/shopify/setup-store-details.png
- assets/shopify/pages-open-pages.png
- assets/shopify/themecustomizer-customizer.png
- assets/shopify/products-draft.png
- assets/shopify/navigation-add-link.png
- assets/shopify/launch-payments.png
- assets/shopify/devsetup-vs-code.png
- assets/shopify/cliinstall-install-cli.png
- assets/shopify/cliworkflow-preview.png
- assets/shopify/projectfiles-install.png
- assets/shopify/devtroubleshooting-versions.png
- assets/shopify/githubsetup-ask-first.png
- assets/shopify/githubworkflow-commit.png

If a screenshot is missing, the module keeps showing the designed placeholder.
